# Couple's Challenge Game

A fun competitive game for couples with 10 challenges split into two phases.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Game Structure

- **Phase 1: At Home** - 5 challenges
- **Phase 2: At the Mall** - 5 challenges

Each challenge allows you to award a point to either player, and automatically advances to the next game. After all 10 games, view the final results with a complete summary.
